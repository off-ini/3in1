****************  Apple Catcher ****************
    
    == Contrôles ==

- Utiliser les touches de direction gauche et droite pour déplacer le personnage
- La touche Esc permet de revenir au menu

****************  Brick Break ****************
    
    == Contrôles ==

- La touche Espace permet de lancer le balle
- Utiliser les touches de direction gauche et droite pour controler le Paddle
- La touche Z permet de lancer un missile

    == Note ==

- Vous avez trois vies au début de chaque partie. A chaque fois que vous manquez de rattraper la balle vous perdez une vie.
- Lors ce que vous n'avez plus de vie vous retournez au menu (Game Over).
- Lors ce qu'il n'y plus de brick vous passez au niveau suivant.
- Vous pouvez ramasser les bonus ou les missile que vous débloquer.

****************  Flappy Brid ****************
    
    == Contrôles ==

- La touche Espace ou le clique gauche de la sourie permettent de controler Flappy

    == Note ==

- plus vous avancez plus vous augmenter vos chance de vous faire atomiser.
- Vous pouvez ramasser les bonus.
